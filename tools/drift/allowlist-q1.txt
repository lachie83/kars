# Wave 4 (commit a277a14) — namespace-resolution fix:
#   super::handoff::MAX_BLOB_SIZE_BYTES  →  handoff::MAX_BLOB_SIZE_BYTES
# At routes.rs top-level, `super` = crate root, so `super::handoff` resolved
# to `crate::handoff`. After moving into routes/handoff.rs, `super::handoff`
# would recurse to the enclosing module itself. The local `use crate::handoff;`
# makes `handoff::X` resolve to `crate::handoff::X` — same constant,
# byte-identical runtime behavior.
